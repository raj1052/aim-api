AIM API

Version: v1 ¶

This API documentation is given details about the all AIM API list. It is basically a reference documents which describes:

  • A Software’s code

  • The parameters a programmer sends (or the request sent) to the software.

  • What the software does

  • Response from the software

API documentation includes information about the code. It gives details about the APIs (also called methods and functions) that developers use to connect or interface with the software application.
